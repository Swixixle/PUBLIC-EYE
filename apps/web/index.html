Frame

Frame turns a political claim into a cryptographically signed receipt: money, votes, and lobbying citations are listed with neutral narrative sentences, each tied to a source id.

Rules

  • Every narrative sentence must include a sourceId from sources.
  • No judgment adjectives in narrative (see governance list in the repo).
  • Hashing uses JCS (RFC 8785) via the canonicalize package — not JSON.stringify.

API

Run the FastAPI app from apps/api, then ping health or verify a receipt.
